# Project Skills

These skills are repo-local reference guides for working on `molstar-structure-editor`.

- `molstar-structure-editor-dev`
  Use when modifying the plugin architecture, editing behavior, or extending editing features.
- `molstar-structure-editor-debug`
  Use when gizmo interaction, coordinate updates, or Mol* rendering behaves unexpectedly.
- `molstar-structure-editor-verify`
  Use before claiming the plugin change is complete, fixed, or ready to commit.

All skills live under `skills/<skill-name>/SKILL.md`.
